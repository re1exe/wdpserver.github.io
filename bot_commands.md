# Here goes commands for the discord bot:

```py

- /addchange: Add a change log entry

- /help: Lists all available commands.

- /ip: Shows the IP of the Minecraft server.

- /message: Sends a message or custom embed.

- /ping: Replies with bot latency.

- /report: Report a player to the staff.

- /rules: Displays Minecraft + Discord rules.

- /status: Check if the Minecraft server is online.

- /suggest: Send a suggestion to the staff.

- /uptime: Shows how long the bot has been running.


```
